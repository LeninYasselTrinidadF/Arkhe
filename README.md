# Arkhe

**Arkhe** is a visual explorer and editor for formal mathematical structures. The project combines the precision of formal logic with a dynamic interface based on particle physics and graphs, utilizing a high-performance rendering engine to navigate through complex taxonomies.

## Key Features

### Multimodal Navigation
Optimized node visualization in three primary modes:
* **MSC2020**: Exploration of research areas based on the **AMS** (Mathematics Subject Classification).
* **Mathlib**: Navigation through the hierarchy of files and declarations of the **Lean4** formal library.
* **Standard**: Customized academic syllabi that connect concepts across various libraries.

### Integrated Entry Editor
Allows for real-time management and editing of data associated with each mathematical node within the graph.

## Project Structure

The code is organized modularly to facilitate system scalability:

* **`/data`**: Specialized loaders for MSC, Mathlib, and cross-reference systems.
* **`/ui`**: User interface components, including the Toolbar, search panels, and information panels.
* **`/search`**: Logical search engines, including integration with tools like **Loogle**.
* **`/assets`**:
    * **`/graphics`**: Storage for system sprites and icons.
    * **`/entries`**: Repository for `.tex` files and detailed descriptions generated by the editor.
